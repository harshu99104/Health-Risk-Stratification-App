# 📌 Project Overview

Health Risk Stratification is a Machine Learning application that predicts a patient's health risk category based on demographic information, laboratory results, diagnoses, and clinical history.

The project combines **Data Analytics**, **Machine Learning**, and **Interactive Visualization** using Streamlit to help demonstrate how predictive analytics can support healthcare decision-making.

# 🚀 Features

✅ Predict patient health risk instantly

✅ Interactive Streamlit dashboard

✅ Machine Learning prediction using Random Forest

✅ Clean and user-friendly interface

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/harshu99104/Health-Risk-Stratification-App.git
```

Move into the project folder

```bash
cd Health-Risk-Stratification-App
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Streamlit app

```bash
streamlit run App.py
```
